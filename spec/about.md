
这个网站是使用 **Astro** 框架，搭配 [Mizuki](https://github.com/matsuzaka-yuki/mizuki) 主题构建的。

::github{repo="matsuzaka-yuki/Mizuki"}

## 🌟 主题特色

### 🎨 设计与用户体验
- **现代而优雅的设计** - 简洁、极简的界面，配有漂亮的排版
- **完全响应式** - 针对从移动设备到桌面的所有设备进行了优化
- **深色/浅色模式** - 自动切换主题并带有平滑过渡
- **精美排版** - 采用 JetBrains Mono 字体增强可读性
- **流畅动画** - 流畅的页面过渡和交互元素

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 驱动，实现快速、准确的结果
- **增强型 Markdown** - 扩展语法，支持代码高亮和数学支持
- **交互式目录** - 针对长文章的自动滚动导航
- **RSS 订阅生成** - 通过自动生成订阅源保持更新
- **阅读时间估算** - 了解文章需要多长时间阅读
- **文章分类** - 使用标签和分类组织内容

### 📱 特殊页面
- **动漫追踪页** - 追踪您的动漫观看进度和评分
- **友情链接页** - 以精美卡片展示朋友的网站
- **日记/瞬间页** - 分享生活瞬间，类似于社交媒体帖子
- **归档页** - 所有文章按时间轴组织的视图
- **关于页** - 可定制的个人介绍（即本页面！）

### 🛠 技术特色
- **增强型代码块** - 由 [Expressive Code](https://expressive-code.com/) 驱动
- **数学支持** - 使用 KaTeX 进行 LaTeX 数学表达式渲染
- **图片优化** - 带有懒加载的 PhotoSwipe 相册
- **SEO 优化** - 内置站点地图和元标签，以获得更好的搜索可见性
- **性能优化** - 通过缓存和优化实现快速加载
- **评论系统就绪** - 支持集成 Twikoo 评论

### 🎯 高级 Markdown 特性
- **标注与提示框** - 漂亮的提示信息框，使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]`
- **数学公式** - 使用 `$inline$` 和 `$$block$$` 语法书写 LaTeX 数学
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 带有行号的高级代码高亮
- **复制代码按钮** - 简便的代码复制功能
